==Event==
(Client) OnPreFillInventoryObjectContextMenu

==Description==
Fires when the context menu for an inventory item is created, before it is filled.

==Parameters==
*playerNum: integer - The number of the player whose context menu has been created.
*context: [[ISContextMenu]] - The context menu that was created.
*items - The items that were selected to fill the context menu. If only full stacks are selected, a table of ContextMenuItemStacks is passed. Otherwise it is a table of InventoryItems.
**InventoryItem[]
**ContextMenuItemStack[]
